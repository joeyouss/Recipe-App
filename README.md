# Recipe-App

Recipe app allowing users to create recipes, edit recipes and delete recipes !

App in action - [Recipe App](https://bubble-maize-bard.glitch.me/)


## Running locally :
- Fork the repo
- Clone the repo
- ```npm i``` to install every dependency
- ```npm run dev``` to start the app

## Tech stack:
- Javascript
- Node.js
- HTML, CSS
- MongoDB
